# CampusHub - Student Resource Portal

## Project Description

CampusHub is a simple web application that helps students access study resources,
view subjects, upload notes, and search academic materials.

The project is developed using:

- Frontend : HTML, CSS, Bootstrap
- Backend : Python Flask
- Database : MySQL
- Version Control : Git & GitHub

# Users

## Student
- Login
- View Subjects
- Search Resources
- Upload Notes

## Faculty
- Add Resources
- Update Resources

## Admin
- Manage Students
- Manage Faculty
- Manage Resources

## Functional Requirements

Student can:

- Login
- View Subjects
- Search Resources
- Upload Notes

Faculty can:

- Add Resources
- Update Resources

Admin can:

- Manage Students
- Manage Faculty
- Manage Resources

## Non Functional Requirements

- Simple UI
- Fast Performance
- Responsive Design
- Easy Navigation
